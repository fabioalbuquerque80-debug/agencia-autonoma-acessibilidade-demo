O que a Agência Autônoma coleta

Esta demonstração não possui formulário, conta, analytics, marketing, upload ou banco de visitantes. A Agência Autônoma não recebe nome, e-mail, URL para auditoria ou conteúdo enviado por você nesta página.

Dados técnicos da hospedagem

Quando publicada no GitHub Pages, a infraestrutura do GitHub pode processar dados técnicos e de uso, como endereço IP, dispositivo, data e hora, páginas vistas e links acessados. Esse tratamento é realizado pelo provedor conforme sua própria política.

Leia a declaração de privacidade do GitHub.

Links externos

Os links para W3C, Governo Digital e GitHub levam a serviços de terceiros, sujeitos às políticas desses sites.

Contato e direitos

Nenhum canal de contato é publicado nesta validação inicial. Antes de qualquer coleta futura, esta página deverá ser atualizada com finalidade, base legítima, retenção, responsável e canal de direitos.
